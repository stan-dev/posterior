# List of available convergence diagnostics

A list of available diagnostics and links to their individual help
pages.

## Value

See individual functions for a description of return types.

|                                                                                   |                                                       |
|-----------------------------------------------------------------------------------|-------------------------------------------------------|
| **Function**                                                                      | **Description**                                       |
| [`ess_basic()`](https://mc-stan.org/posterior/dev/reference/ess_basic.md)         | Basic version of effective sample size                |
| [`ess_bulk()`](https://mc-stan.org/posterior/dev/reference/ess_bulk.md)           | Bulk effective sample size                            |
| [`ess_tail()`](https://mc-stan.org/posterior/dev/reference/ess_tail.md)           | Tail effective sample size                            |
| [`ess_mean()`](https://mc-stan.org/posterior/dev/reference/ess_mean.md)           | Effective sample sizes for the mean                   |
| [`ess_median()`](https://mc-stan.org/posterior/dev/reference/ess_quantile.md)     | Effective sample sizes for the median                 |
| [`ess_quantile()`](https://mc-stan.org/posterior/dev/reference/ess_quantile.md)   | Effective sample sizes for quantiles                  |
| [`ess_sd()`](https://mc-stan.org/posterior/dev/reference/ess_sd.md)               | Effective sample sizes for the standard deviation     |
| [`mcse_mean()`](https://mc-stan.org/posterior/dev/reference/mcse_mean.md)         | Monte Carlo standard error for the mean               |
| [`mcse_median()`](https://mc-stan.org/posterior/dev/reference/mcse_quantile.md)   | Monte Carlo standard error for the median             |
| [`mcse_quantile()`](https://mc-stan.org/posterior/dev/reference/mcse_quantile.md) | Monte Carlo standard error for quantiles              |
| [`mcse_sd()`](https://mc-stan.org/posterior/dev/reference/mcse_sd.md)             | Monte Carlo standard error for the standard deviation |
| [`pareto_khat()`](https://mc-stan.org/posterior/dev/reference/pareto_khat.md)     | Pareto khat diagnostic for tail(s)                    |
| [`pareto_diags()`](https://mc-stan.org/posterior/dev/reference/pareto_diags.md)   | Additional diagnostics related to Pareto khat         |
| [`rhat_basic()`](https://mc-stan.org/posterior/dev/reference/rhat_basic.md)       | Basic version of Rhat                                 |
| [`rhat()`](https://mc-stan.org/posterior/dev/reference/rhat.md)                   | Improved, rank-based version of Rhat                  |
| [`rhat_nested()`](https://mc-stan.org/posterior/dev/reference/rhat_nested.md)     | Rhat for use with many short chains                   |
| [`rstar()`](https://mc-stan.org/posterior/dev/reference/rstar.md)                 | R\* diagnostic                                        |
