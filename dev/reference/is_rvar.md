# Is `x` a random variable?

Test if `x` is an
[`rvar`](https://mc-stan.org/posterior/dev/reference/rvar.md).

## Usage

``` r
is_rvar(x)
```

## Arguments

- x:

  (any object) An object to test.

## Value

`TRUE` if `x` is an
[`rvar`](https://mc-stan.org/posterior/dev/reference/rvar.md), `FALSE`
otherwise.

## See also

[`as_rvar()`](https://mc-stan.org/posterior/dev/reference/as_rvar.md) to
convert objects to `rvar`s.
